## 搬瓦工 IP 被封了，先别急着换——先检测

**搬瓦工 IP 被封** 是指你的 VPS 主 IP 被加入了中国大陆的黑名单，从国内无法访问，但服务器本身运行正常，从海外访问依然通。

换 IP 之前，建议先用搬瓦工官方提供的检测工具确认一下。

检测步骤：

1. 登录搬瓦工客户中心，进入 Services → My Services，找到对应 VPS
2. 点击「KiwiVM Control Panel」进入后台管理面板
3. 登录 KiwiVM 之后，在浏览器地址栏输入以下链接：


https://kiwivm.64clouds.com/main-exec.php?mode=blacklistcheck


4. 点击「Test Main IP」，等待检测结果
5. 显示 **IP NOT BLOCKED** → IP 正常，不需要换
6. 显示 **IP BLOCKED** → 确认被封，按下面的方法处理

有一点要注意：有时候连不上是因为端口被封，不是 IP 的问题。端口被封不需要换 IP，只需要换一个未被封的端口就行。所以检测这一步不要跳过。

## IP 被封之后，几个常见误区

**误区一：重装系统能换 IP？**

不能。重装系统只是把操作系统重新装一遍，IP 地址不变，封的还是封。

**误区二：VPN 套 VPN 能绕过 IP 被封？**

如果 IP 本身在黑名单里，国内流量根本到不了你的服务器，套什么都没用。

**误区三：等一段时间 IP 会自动解封？**

理论上 IP 可能被解封，但时间不确定，少则几周，多则几个月，等不起的话还是直接换。

**误区四：换 IP 之后之前的配置要重新来？**

IP 变了，SSH 连接地址要更新，域名解析如果绑定了旧 IP 也要改。但服务器里的文件、程序、配置都不受影响，换完 IP 改一下连接信息就好。

**Q：搬瓦工 IP 被封后还能切换机房吗？**

**Q：换 IP 之后新 IP 会不会很快又被封？**

搬瓦工换的 IP 官方保证全球可达，从国内也能用。后续会不会被封取决于使用方式，正常建站、跑服务不太会有问题。

付费申请之后，等待时间最长 24 小时，通常客服处理比较快，有时候几个小时就好了。换完会在 KiwiVM 面板和工单里通知你新 IP。
